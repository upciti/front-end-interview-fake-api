# Upciti front-end interview fake API

This project is used solely for the front-end interview challenge purpose and to build a Docker image.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Run tests

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
